## 四、数据库配置

### 1. 创建集合
在云开发控制台中创建以下集合：

- [ ] teachers（教员接单表）
- [ ] users（用户表）
- [ ] sms_codes（验证码表）
- [ ] counters（计数器表）

### 2. 初始化计数器
在counters集合中添加初始数据：

```json
{
  "_id": "teacherId",
  "count": 100000
}
```

操作步骤：
- [ ] 进入counters集合
- [ ] 点击"添加记录"
- [ ] 输入上述JSON数据
- [ ] 保存

### 3. 配置数据库权限
在云开发控制台中配置各集合的权限：

**teachers集合：**
- [ ] 读权限：所有用户可读
- [ ] 写权限：仅创建者可写

**users集合：**
- [ ] 读权限：仅创建者可读
- [ ] 写权限：仅创建者可写

**sms_codes集合：**
- [ ] 读权限：仅创建者可读
- [ ] 写权限：所有用户可写

**counters集合：**
- [ ] 读权限：所有用户可读
- [ ] 写权限：仅管理员可写

### 4. 创建索引
按照 `DATABASE_INDEX_GUIDE.md` 中的说明创建索引：

- [ ] teachers.teacherId（升序）
- [ ] teachers.openid（升序）
- [ ] teachers.status（升序）
- [ ] teachers.school（升序）
- [ ] teachers.createTime（降序）
- [ ] teachers.status + createTime（复合索引）
- [ ] users.openid（升序）
- [ ] users.phone（升序）
- [ ] sms_codes.phone（升序）
- [ ] sms_codes.expireTime（升序）

## 常见问题排查

### 问题1：云函数调用失败
**可能原因：**
- 云函数未上传部署
- 云环境ID配置错误
- 云函数代码有语法错误

**解决方法：**
1. 检查云开发控制台，确认云函数状态
2. 检查app.js中的云环境ID
3. 查看云函数日志，定位错误

### 问题2：数据库查询返回空
**可能原因：**
- 集合未创建
- 权限配置错误
- 查询条件有误

**解决方法：**
1. 确认集合已创建
2. 检查数据库权限配置
3. 在控制台测试查询语句

### 问题3：图片无法显示
**可能原因：**
- 文件路径错误
- fileID错误
- 图片未上传

**解决方法：**
1. 检查图片是否存在于指定路径
2. 确认云存储fileID正确
3. 检查云存储权限配置

### 问题4：TabBar图标不显示
**可能原因：**
- 图标文件缺失
- 路径配置错误
- 图标尺寸不正确

**解决方法：**
1. 确认图标文件存在于images目录
2. 检查app.json中的iconPath配置
3. 确认图标尺寸为81×81px
